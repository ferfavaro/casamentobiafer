# Site de Casamento - Beatriz & Fernando

Projeto estático em React + TypeScript inspirado no layout enviado.

## Como usar

1. Instale dependências:

```bash
npm install
```

2. Rode localmente:

```bash
npm run dev
```

3. Build de produção:

```bash
npm run build
```

## Ajustes importantes

- Substitua o arquivo `public/background-casal.jpg` pela segunda foto enviada (do casal em frente à igreja).
- Atualize os endpoints:
  - `RSVP_POST_URL` em `src/App.tsx`
  - `GIFT_POST_URL` em `src/App.tsx`
- Atualize `validInviteCodes` com os códigos reais dos convites.
- Personalize a lista `gifts` com os presentes finais e suas descrições.
